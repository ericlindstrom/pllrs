PLLRS

Responsive CSS Column Structure

What is it?

PLLRS is a library for responsive uniform column structure. Built considering mobile first, It handles 4 breakpoints with media queries. By default, they are:

  • base (mobile first)
  • 600px (small tablet)
  • 768px (large tablet)
  • 1000px (desktop)

How Do I use it?

You set your class pllrs4 and dictate how your breakpoints fold with a secondary class of underscores __2_

A 4 Column

What does it look like?

<div class="_column3 ____">
  <div class="column"></div>
  <div class="column"></div>
  <div class="column"></div>
  <div class="column"></div>
  <div class="column"></div>
</div>
